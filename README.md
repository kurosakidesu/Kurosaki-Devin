# 陸上入出荷オーダ一覧画面

Excel画面設計書（OFFSITE-V-0201）に基づく「陸上入出荷オーダ一覧画面」の実装です。

## 技術スタック

- **バックエンド**: Python FastAPI + SQLite
- **フロントエンド**: HTML / CSS / JavaScript（バニラ）

## 起動手順

### 1. 必要環境

- Python 3.9 以上

### 2. バックエンド起動

```bash
cd backend
pip install -r requirements.txt
python main.py
```

バックエンドが `http://localhost:8000` で起動します。
起動時にSQLiteデータベース（`land_order.db`）が自動作成され、初期データが投入されます。

### 3. フロントエンド表示

`frontend/index.html` をブラウザで直接開くか、簡易サーバーを使用します。

```bash
cd frontend
python -m http.server 3000
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## DB テーブル

| テーブル名 | 説明 |
|-----------|------|
| land_order | 陸上オーダ |
| land_record | 陸上実績 |
| m_shipping_sts | 出荷ステータスマスタ |
| m_code_name_pair | 名称・区分マスタ |
| lane_situ | レーン利用状況 |

## サンプルデータ

起動時に20件以上のサンプルデータが自動投入されます。
- ステータス違い（予約〜実績送信完了）
- 日付違い（前日・当日・翌日）
- マスタ未設定・空値
- 長文テスト
