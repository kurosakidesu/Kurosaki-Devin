# Devin検証メモ（移送オーダ一覧 OFFSITE-V-0401）

## ■ 概要
- Excel画面設計書を元にDevinで業務画面を実装
- テスト実施および報告書作成まで一連の流れを検証
- プロンプトは段階的に投入（実装 → テスト → 修正）

---

## ■ 1回目（実装）

### 制約
- 単一画面のみ実装（他画面禁止）
- 画面遷移はダミー
- 業務機能の追加は禁止
  - 新しいステータス
  - 新しい検索条件
  - 新しいボタン
  - 承認 / 通知 / バッチ / 帳票 / 外部連携

### データ
- SQLiteの仮DBを使用
- 初期データ投入あり（約20件）
- 以下を含める
  - ステータス違い
  - 日付違い
  - 空値
  - 長文
  - マスタ未設定
  - 0件検索条件

### 成果物
- フロントエンド
- バックエンドAPI
- DB / seedデータ
- README

### 結果
- 指示通り生成（暴走なし）
- UI / API / DBが一貫して生成
- UIはやや古めの業務画面

---

## ■ 技術スタック
- Frontend: React + TypeScript + Vite
- Backend: FastAPI
- DB: SQLite
